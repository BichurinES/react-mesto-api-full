# Проект - Mesto

## Описание проекта
Проект представляет собой журнал путешественника, с возможностью добавления новых мест, после их посещения.

Ориентирован на просмотр с любого устройства.

Ссылка на проект: **https://mestogram.nomoredomains.monster/**

## Используемые технологии:

__FRONTEND__
* __React__
  * Проект разбит на функциональные компоненты
  * Функционал регистрация и входа по логину и паролю, реализован через http-only куки для большей безопасности
  * Реализован функционал live-валидации форм со стороны клиента
  * Защищенные роуты для доступа к контенту только зарегистрированных пользователей
  * Реализован функционал кнопки редактирования профиля
  * Реализован функционал формы обновления данных об имени и описании пользователя
  * Реализован функционал добавления своего места
  * Реализован функционал увеличения фото по клику
  * Используются хуки для работы с эффектами
  * Используется декларативный подход в разработке
* __HTML5__
  * Семантические тэги
  * Валидная разметка
* __CSS__
  * Адаптивный сайт под все возможные экраны
  * Сэндвич-меню, трансформация для небольших экранов
  * Построение сеток с помощью flex и grid
  * Позиционирование сложных элементов
  * Реализован функционал "живой" валидации форм со стороны клиента
* __БЭМ__
  * Все классы названы в соответствии с БЭМ
  * Создана файловая структура по модели nested, все блоки разбиты по отдельным папкам, элементы и модификаторы находятся в папках блоков по собственным разделам

__BACKEND__
* __Express__
  * Прописана логика сервера для работы с регистрацией и входа по логину и паролю
  * Реализована логика обработки CORS-запросов
  * Автоматическая проверка введенных данных на корректность через библиотеку celebrate
  * Единый центр обработки всех ошибок
  * Добавление, удаление мест, простановка и снятие лайка
  * Редактирование профиля пользователя
  * Модели и связка с базой данных MongoDB
